# 5주차 실습 기록

## 선택한 템플릿 및 주제
- **템플릿:** Space Shooter  
- **주제:** 거대한 보스를 상대하는 2D 액션 슈터 게임 "부숴버려라"  
- 목표는 플레이어가 WASD로 이동하고 마우스 방향으로 총알을 발사하며 보스를 처치하는 것.

## 사용한 AI 프롬프트 (요약)
1. 게임 코드 전체 구조 설명 요청  
2. 기획서 초안을 기반으로 구체적 규칙, 난이도, 승패 조건 등 보완 요청  
3. 3주 안에 구현 가능한 규모인지 확인 요청  
4. 보스 패턴, 돌기 공격, 타격 및 무적 시스템 구현 방법 안내 요청  

## AI 답변에서 도움이 된 것
- Python `pygame` 구조를 이해하고 기존 코드 개선 방향을 잡는 데 도움  
- 기획서에서 난이도 단계, 공격 패턴, 플레이어 조작 방법 등을 구체화  
- 개발 소요 시간과 단계별 구현 계획을 제시받아 현실성 평가 가능  
- 보스/돌기 공격, 피격 이펙트, 총알 방향 회전 등 핵심 구현 아이디어 제공  

## AI 답변을 수정하거나 버린 것
- 일부 공격 패턴과 레이저 패턴은 실제 구현 시 복잡해 불필요하여 삭제  
- 보스 크기, 돌기 위치, 피격/무적 타이밍 등 디테일 수정  
- 플레이어 HUD 위치, 총알 미리 생성되는 버그 등 AI 제안 코드는 버림  
- 최종적으로 게임 난이도와 공격 패턴은 직접 조정하여 안정적 플레이 확보
